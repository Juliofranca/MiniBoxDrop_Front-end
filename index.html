Sobre

Mini BoxDrop Application 1.0.0

A aplicação foi desenvolvida para gerenciar produtos. O sistema permite o cadastro, edição. O sistema usa HTML, CSS, Bootstrap, JavaScript e Bootstrap. A idéia principal do sistema é permitir que o usuário possa cadastrar produtos, editar e excluir produtos com anexos. Assim, um usuário pode ter um controle maior sobre os produtos que ele tem em estoque. O sistema pode ser usado para hospedar cursos online, aulas, palestras, seminários, etc. Após o cadastro, o usuário pode fazer o upload de arquivos de vídeo, áudio, pdf, etc. O sistema também permite que o usuário faça o download dos arquivos que ele enviou.
